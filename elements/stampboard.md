# `<stampboard />`

Board with castellated stamp-style edges.

## Example

```tsx
export default () => (
  <stampboard width="21mm" height="18mm" leftPinCount={10} rightPinCount={10} pinPitch="2.54mm" />
)
```

## Props

Commonly used: `leftPinCount`, `rightPinCount`, `topPinCount`, `bottomPinCount`, `leftPins`, `rightPins`, `topPins`, `bottomPins`
